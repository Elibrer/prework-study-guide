Kek Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use "h1", "h2" etc.. to create various sized headers.
  • Use src= with a link to include an image/video - pulling it from the source.
  • ul, ol and li is used for unordered lists, ordered lists, and simply lists.
  • p is used to create a paragraph.
  • br is used for a break in the page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Refer to style.css for various fomartting commands and tips.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A Add changed files from working directory to the staging area of Git.
  • git commit -m "Insert change notes here" Commit the changes to the repository.
  • git pull origin main Use this to pull in the base branch.
  • git push origin feature/*Whatever the feature name is* Use this to push the changes to GitHub.
  • git checkout main Returns the user back to the main branch.
